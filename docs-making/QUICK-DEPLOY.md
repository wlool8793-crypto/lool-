## **Method 1: GitHub Pages (Easiest - 5 Minutes)**

### **Step 1: Create GitHub Account**
Go to [github.com](https://github.com) and sign up (free)

### **Step 2: Create New Repository**
1. Click "+" ‚Üí "New repository"
2. Name: `google-docs-clone`
3. Set to "Public"
4. Click "Create repository"

### **Step 3: Upload Your Files**
1. Click "uploading an existing file"
2. Drag and drop ALL files from your `docs-making` folder:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `CLAUDE.md`
   - `README.md`
   - `.nojekyll`
   - All other files

3. Write commit message: "Initial deploy"
4. Click "Commit changes"

### **Step 4: Enable GitHub Pages**
1. Go to **Settings** (in your repo)
2. Scroll down to **GitHub Pages** section
3. Under **Source**, select **Deploy from a branch**
4. Choose **Branch:** `main`
5. Choose **Folder:** `/ (root)`
6. Click **Save**

### **Step 5: Your Site is LIVE!**
Wait 2-3 minutes, then visit:
`https://YOUR_USERNAME.github.io/google-docs-clone/`

## **Method 2: Netlify (Drag & Drop - 2 Minutes)**

### **Step 1: Go to Netlify**
Visit [netlify.com](https://netlify.com) and sign up (free)

### **Step 2: Drag & Drop**
1. Click "Deploy manually" or "Drag and drop"
2. Drag your ENTIRE `docs-making` folder onto the page
3. Wait for upload to complete

### **Step 3: Your Site is LIVE!**
Netlify will give you a random URL like:
`https://amazing-tesla-123456.netlify.app`

## **Method 3: Vercel (Command Line - 3 Minutes)**

### **Step 1: Install Vercel**
```bash
npm install -g vercel
```

### **Step 2: Deploy**
```bash
cd docs-making
vercel
```

### **Step 3: Follow Prompts**
- Enter your email
- Choose project settings
- Your site will be deployed instantly!
